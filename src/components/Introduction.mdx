## Available Components

### Chat Interface
- **Chat Container**: The main container component that orchestrates the chat interface layout and interactions.
- **Chat Bubble**: Stylish message bubbles for displaying user and AI messages with support for various content types.
- **Chat Input**: Feature-rich input component with support for text, commands, and file attachments.

### AI Interaction
- **AI Processing Indicator**: Visual feedback component showing when the AI is processing or generating responses.
- **Model Selector**: Component for selecting and configuring different AI models.
- **Reasoning Toggle**: Toggle component for displaying AI reasoning and thought processes.

### Navigation & Context
- **Chat Selector**: Navigation component for switching between different chat conversations.
- **Chat Suggestions**: Dynamic suggestion chips to help users with common actions or responses.
- **Context List**: Display and manage contextual information used in the chat.

### File Management
- **File Selector**: File management component for uploading and managing chat attachments.
- **Avatar Holder**: Component for displaying user and AI avatars consistently.
- **Selected Files**: Display and manage currently selected files in the chat.

## Getting Started

To start using ChatKit in your project:

```bash
npm install @thexeroxbe/chat-ui-react
```

Check out our component documentation in the sidebar to see detailed usage examples and API references for each component.

## Customization

All components support theme customization through our theme provider. You can modify colors, spacing, typography, and other design tokens to match your application's design system.
