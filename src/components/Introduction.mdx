## Getting Started

To use these components in your project:

```tsx
import { ThemeProvider, ChatBubble } from 'chat-ui';

// Optional: Create a custom theme
const customTheme = {
  colors: {
    userBubbleBackground: '#2E5CD3',
    // ... other theme overrides
  }
};

function App() {
  return (
    <ThemeProvider theme={customTheme}>
      <ChatBubble 
        message="Hello World!"
        isUser={false}
        timestamp="12:30 PM"
      />
    </ThemeProvider>
  );
}
```

## Available Components

### ChatBubble
A versatile chat bubble component that supports:
- User and AI messages with different styling
- Timestamps
- Extra content slot for AI model indicators or status information
- Customizable appearance through theming
