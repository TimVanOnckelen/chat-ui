## Available Components

### Core Chat Components
- **ChatBubble**: Message bubbles for user and AI responses with timestamp and extra content support
- **ChatContainer**: Layout container for chat messages with auto-scrolling and message grouping
- **ChatInput**: Text input component with file attachment and submission handling
- **AIProcessingIndicator**: Loading indicator for AI response generation

### Selection Components
- **ChatSelector**: Switch between different chat sessions or conversations
- **ModelSelector**: Choose between different AI models or configurations
- **FileSelector**: File upload and attachment handling component
- **SelectedFiles**: Display and manage selected file attachments

## Getting Started

To use these components in your project:


## Theming

The library uses a theme provider that allows you to customize:
- Colors (backgrounds, text, accents)
- Spacing and layout
- Typography (font families, sizes, weights)
- Border radius and shadows
- Component-specific styles

### Example Theme Customization

```tsx
const customTheme = {
  colors: {
    primary: '#2E5CD3',
    background: '#FFFFFF',
    userBubbleBackground: '#2E5CD3',
    aiBubbleBackground: '#F0F0F0',
    text: '#1A1A1A'
  },
  spacing: {
    small: '8px',
    medium: '16px',
    large: '24px'
  }
};

<ThemeProvider initialTheme={customTheme}>
  {/* Your app content */}
</ThemeProvider>
```
